FORMATIONPROT - SITE HTML PRO

COMMENT VOIR LE SITE
1. Dezippez le dossier.
2. Double-cliquez sur index.html.
3. Le site marche sans Node, sans npm, sans serveur.

COMMENT MODIFIER
- Texte principal : index.html
- Couleurs/design/animations : assets/style.css
- Menu, slider, filtres, WhatsApp : assets/script.js

IMPORTANT
Remplacez partout : 216XXXXXXXX par votre numero WhatsApp.
Exemple Tunisie : 21612345678

AJOUTER UNE DEMO
1. Ouvrez index.html.
2. Cherchez : <section id="demos"
3. Copiez une carte : <article class="project reveal" ...> ... </article>
4. Modifiez :
   - data-category="web" ou "app" ou "marketing" ou "formation"
   - le titre <h3>
   - la description <p>
   - le lien href="demos/mon-projet.html"
5. Dans le dossier demos, copiez une page existante et renommez-la mon-projet.html.
6. Modifiez son contenu.

HEBERGEMENT GRATUIT
Option simple : Netlify Drop ou Cloudflare Pages en upload direct.
Uploadez tout le dossier, pas seulement index.html.
